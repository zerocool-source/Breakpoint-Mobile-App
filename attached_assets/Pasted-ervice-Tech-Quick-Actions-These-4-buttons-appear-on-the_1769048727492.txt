These 4 buttons appear on the Property screen and allow technicians to quickly report issues or request items while on-site.

1. üîß Repairs Needed (Red Wrench)
Purpose: Report equipment or property issues that need repair attention

When tapped, opens a form with:

Field	Type	Options/Details
Issue Type	Dropdown	Pump problem, Filter issue, Heater malfunction, Leak detected, Electrical issue, Other
Location	Auto-filled	Current property name
Description	Text area	Describe what's wrong
Priority	Dropdown	Low, Normal, High, Urgent

Equipment Tag	Text input	Equipment ID if visible
What happens: Sends repair request to office/repair team for scheduling

Quantity	Number inputs	Amount for each selected chemical
Urgency	Dropdown	Routine, Next Visit, ASAP
Notes	Text area	Special instructions
What happens: Sends order to warehouse/chemical delivery team
